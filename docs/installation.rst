============
Installation
============

Obsea is a pure Python package published on PyPI_. It requires Python 3.10 or later and can be installed, along with all its dependencies, with::

    pip install obsea

Note that some dependencies (e.g. ObsPy_) recommend installation with ``conda``. Obsea works either way: it can be pip-installed in a conda environment as well. This is up to the user.

.. _PyPI: https://pypi.org/project/obsea/
.. _ObsPy: https://docs.obspy.org/
